Guide to Simpris

Concepts

In Simpris you start with a project, divide it into tasklists, and divide each tasklist into tasks.

  • Project 1
    • 1st Tasklist
      • Task 1
      • Task 2
    • 2nd Tasklist
      • Task A
      • Task B
      • Task C
      • Task D

You can create many projects, and each project can have many tasklists, and each tasklist can have many tasks.

Projects can also be organised into programmes:

  • Programme 1
    • Project 2
      • 3rd Tasklist
        • Task 3
        • Task 4
      • 2nd Tasklist
        • Task E
        • Task F
        • Task G
        • Task H
    • Project 3
      • 4th Tasklist
        • Task 5
        • Task 6
      • 5th Tasklist
        • Task I
        • Task J
        • Task K
        • Task L

Finding your way around Simpris

Menus

At the top of Simpris is its menu system, which allows you to navigate around by clicking on its links. The menus link to pages in Simpris which have lists of projects, tasks and other information.

In each list page the following icons appear:

  •  Delete the item
  •  Edit the item
  •  View the item
  •  Any charts or metrics pertinent
  •  Further charts or metrics

The Simpris list pages are as follows:


My Clients

A list of your clients. Clients are organisations or people for whom you are working on your projects. A client may be your employer, your customer, a big organisation or corporation, or a small company.

My Projects

A list of your projects. These are the projects which you have been granted permission to see and work on.


My Tasks

A list of your tasks. These are the tasks which you have been granted permission to see and work on.


My Problems

A list of your problems. These are the problems which you have been granted permission to see and work on.


People

A list of people from your organisation and clients.


My Time

A list of the time you have spent on various tasks. Here you can also book time to tasks.


Administration

Functions for system administrators. You will only be able to use these if you have been made a system administrator.

Setting up a project

To begin using Simpris you must create your project. Hover your mouse over the 'My Projects' menu and click the 'Add Project' menu. This shows a form as follows:

  • Project name: give it an appropriate name
  • Project description: give your project a full description so other users know its purpose.
  • Project stakeholder: generally the person for whom the project is being carried out. For example in a business reorganisation the stakeholder would be the owner or manager of the business.
  • Project manager: who will be responsible for the running of the project.
  • Deliverables: those things which the project is expected to deliver so for example a construction project would deliver a finished building or modification to an existing building. Enter the details as deliverables so everyone who works on the project knows exactly what is to be delivered.
  • Project budget: this will be total expected cost of the project.
  • Importance: the importance of the project - how critical it is.

When you have completed your project details press the submit button and Simpris will create your project. You can view a list of your projects by taking the 'My problems' menu.

Adding users for your project

The next step is to add users for your project - these are the people who can view and work on the project. If you do not add a particular user they will not be able to see the project or work on its tasks. That way you can manage confidentiality on your projects.

To add a user hover your mouse cursor over the menu 'People' and click the 'Add User' menu. This will show a form to be completed as follows:

  • First name: first name of the user
  • Last name: last name of the user
  • Email: the users email address - Simpris needs this to send emails
  • Password - a password which the user will use to gain access. Make this a secure password.
  • Organisation - the organisation for which the user works or represents

Make sure that you give users permissions on projects by editing the project and adding users to the project. If users are not allocated to a project they cannot work on it.

Adding tasklists for your project

Tasklists are a way of organising your projects into manageable pieces. Think of each tasklist as a unit as a major part of your project. The tasklist itself will be divided into tasks.

To add a tasklist, first go to the 'My projects' page and find your project. Then click on the icon to view the project. Here in the project view page, find and click the link titled 'Add new tasklist for this project'. This takes you to a form titled 'Add tasklist' to be completed as follows:

  • Tasklist name: a short name for your tasklist.
  • Tasklist decription: a full description of your tasklist.

When you have completed the form press the button to add the tasklist.

Adding tasks for your tasklists

Once you have added your projects and tasklists you are ready to add tasks. Tasks can only belong to tasklists so you organise your tasks into these. If a particular task does not fit into your existing tasklists, add a new tasklist first.

Tasks are the basic blocks of Simpris. To add a task, navigate to the appropriate project by taking the 'My Projects' menu and clicking on the icon to view the project. Find the tasklist to which you want to add the task and click on the 'Add new task to this list' link.

  • Task description: a description of the task
  • Task type: select the task type from the drop down menu
  • Task priority: the priority of the task
  • Task status: the status of the task
  • Task assigned to: who the task is assigned to

When you have filled in the form, press the 'Add task' button to create the task

Working with projects, tasklists and tasks

In Simpris the secret is to organisae yoyr tasks into appropriate tasklists and organise those into appropriate projects.

As soon as a task is complete close it by setting its status to complete.

Working with time

Simpris can help you to track how much time you spend on your projects. Time is assigned to tasks only - you must have a task for each type of activity in a project.

To add time against a task hover your mouse cursor over the 'My Time' menu and take the 'Add Time' menu. Complete the form as follows:

  • Date of time: the date for which you wish to record time
  • No of hours to record: how many hours you wish to record.
  • Choose task: choose a task against which to record time
  • Time type: choose the type of task against which to record time
  • Comments: comments about your time

Time editor

The time editor is a quick way to add times for various tasks. To use the time editor:
  • Choose the date for which you wish to add time
  • Choose the project task combination for which you wish to add time. Time is recorded against tasks
  • Choose the time started and duration of the time spent on the task
  • Press the insert button to add the time against the task

Working with invoices

When you have recorded enough time you may wish to generate an invoice. This feature is useful for feeelance staff working on projects. To create an invoice hover your mouse cursor over the 'My time' menu and take the 'Create invoice' menu. This displays a form to be completed as follows:

  • Start date: the start date of the period you want the invoice to cover
  • End date: the end date of the period you want the invoice to cover
  • Description/Reference for Invoice: a reference or dexcription by which you can refer to your invoice. This might be for a related series of tasks or a reference number for a job performed.
  • Invoice comments: any comments pertinent to the invoice.

When you have completed the invoice press the 'Create Invoice' button to take you to the next stage of invoice creation. Simpris now displays the 'Prepare Invoice' page which allows you to allocate tasks to your invoice.

In the 'Prepare Invoice' page you must un-tick any tasks which you do not want to display in your invoice. Then press the 'Add tasks to Invoice' button and your invoice will be created. Simpris then shows you your invoice and you will notice that there is a print button displayed. This will diaplay a printer-friendly version of your invoice.

Working with problems

Any project has its problems and Simpris allows you to track problems against your projects. To create a problem hover your mouse cursor over the 'My Problems' menu and then click the 'Add problem' menu. Simpris will display the 'Add new problem' page with a form to complete as follows:

  • Organisation: the organisation to which the problem applies
  • Problem type: the type of problem
  • Problem subtype: the subtype of problem
  • Problem summary: the summary description of the problem
  • Problem description: full dexription of the problem
  • No of people affected by the problem: an estimate of the number of people affected by the problem
  • Scope: scope of the problem - does it affect a whole organisation or just a few people?
  • Problem Status: status of the problem
  • Problem priority: priority of the problem i.e. is it more or less important

When you have completed the form click the 'Add problem' button to create your problem.

Linking problems

Once you have created your problem you can link it to either a project or another problem.

Link a problem to a project if it pertains to that project

Link a problem to another problem if it should be grouped with that problem. For example one problem - a power failure - might cause and thus be linked to other problems such as office equipment not working.
